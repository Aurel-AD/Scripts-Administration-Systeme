# 🌐 Test-NetworkConnectivity

## 📝 Description
Outil de diagnostic réseau rapide tout-en-un pour le niveau 1 (Helpdesk). Il permet d'isoler rapidement la cause d'une panne réseau ou d'un problème de télétravail chez un utilisateur.

## ⚙️ Prérequis
- Aucun (fonctionne avec les droits utilisateur standards).
- Client VPN (ex: FortiClient, Cisco AnyConnect) si test VPN requis.

## 🚀 Utilisation
```powershell
.\Test-NetworkConnectivity.ps1 -TargetDomain "mondomaine.local"
